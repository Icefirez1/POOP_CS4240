BIN Function.

>>> bin(2)
>>> 0b10
>>> bin(333)
>>> 0b101001101
>>> bin(-42)
>>> -0b101010

the bin() function takes an integer an returns the binary equivalent. It puts a negative sign if the integer is negative

HEX Function.

>>> hex(2)
>>> 0x2
>>> hex(333)
>>> 0x14d
>>> hex(-42)
>>> -0x2a

the bin() function takes an integer an returns the hexadecimal equivalent. It puts a negative sign if the integer is negative

OCT Function.

>>> oct(2)
>>> 0o2
>>> oct(333)
>>> 0o515
>>> oct(-42)
>>> -0o52

the bin() function takes an integer an returns the octal equivalent. It puts a negative sign if the integer is negative
